# Zellij Keybindings

Theme: **Catppuccin Mocha**

## Global (Locked 以外の全モードで有効)

| Key | Action |
|-----|--------|
| `Ctrl g` | Locked モード |
| `Ctrl q` | Zellij 終了 |
| `Ctrl p` | Pane モード |
| `Ctrl t` | Tab モード |
| `Ctrl n` | Resize モード |
| `Ctrl h` | Move モード |
| `Ctrl o` | Session モード |
| `Ctrl s` | Scroll モード |
| `Alt h` / `Alt ←` | フォーカスを左のペイン/タブへ |
| `Alt j` / `Alt ↓` | フォーカスを下のペインへ |
| `Alt k` / `Alt ↑` | フォーカスを上のペインへ |
| `Alt l` / `Alt →` | フォーカスを右のペイン/タブへ |
| `Alt n` | 新規ペイン |
| `Alt f` | フローティングペイン切替 |
| `Alt i` | タブを左へ移動 |
| `Alt o` | タブを右へ移動 |
| `Alt p` | ペイングループ内の切替 |
| `Alt Shift p` | グループマーキング切替 |
| `Alt +` / `Alt =` | リサイズ拡大 |
| `Alt -` | リサイズ縮小 |
| `Alt [` | 前のスワップレイアウト |
| `Alt ]` | 次のスワップレイアウト |
| `Esc` | Normal モードに戻る |
| `Enter` | Normal モードに戻る |

## Locked (`Ctrl g`)

| Key | Action |
|-----|--------|
| `Ctrl g` | Normal モードに戻る |

## Pane (`Ctrl p`)

| Key | Action |
|-----|--------|
| `h` / `←` | フォーカスを左へ |
| `j` / `↓` | フォーカスを下へ |
| `k` / `↑` | フォーカスを上へ |
| `l` / `→` | フォーカスを右へ |
| `n` | 新規ペイン |
| `d` | 新規ペイン (下) |
| `r` | 新規ペイン (右) |
| `s` | 新規ペイン (スタック) |
| `x` | ペインを閉じる |
| `f` | フルスクリーン切替 |
| `w` | フローティングペイン切替 |
| `e` | 埋め込み/フローティング切替 |
| `i` | ペインピン留め切替 |
| `z` | ペインフレーム表示切替 |
| `c` | ペイン名変更 |
| `p` | フォーカス切替 |
| `Ctrl p` | Normal モードに戻る |

## Tab (`Ctrl t`)

| Key | Action |
|-----|--------|
| `h` / `←` / `k` / `↑` | 前のタブ |
| `j` / `↓` / `l` / `→` | 次のタブ |
| `1`-`9` | タブ番号で移動 |
| `n` | 新規タブ |
| `x` | タブを閉じる |
| `r` | タブ名変更 |
| `s` | タブ同期切替 |
| `b` | ペインをタブに分離 |
| `[` | ペインを左のタブに移動 |
| `]` | ペインを右のタブに移動 |
| `Tab` | 前のタブとトグル |
| `Ctrl t` | Normal モードに戻る |

## Resize (`Ctrl n`)

| Key | Action |
|-----|--------|
| `h` / `←` | 左方向に拡大 |
| `j` / `↓` | 下方向に拡大 |
| `k` / `↑` | 上方向に拡大 |
| `l` / `→` | 右方向に拡大 |
| `H` | 左方向に縮小 |
| `J` | 下方向に縮小 |
| `K` | 上方向に縮小 |
| `L` | 右方向に縮小 |
| `+` / `=` | 全体拡大 |
| `-` | 全体縮小 |
| `Ctrl n` | Normal モードに戻る |

## Move (`Ctrl h`)

| Key | Action |
|-----|--------|
| `h` / `←` | ペインを左へ移動 |
| `j` / `↓` | ペインを下へ移動 |
| `k` / `↑` | ペインを上へ移動 |
| `l` / `→` | ペインを右へ移動 |
| `n` / `Tab` | 次の位置へ移動 |
| `p` | 前の位置へ移動 |
| `Ctrl h` | Normal モードに戻る |

## Scroll (`Ctrl s`)

| Key | Action |
|-----|--------|
| `j` / `↓` | 1行下スクロール |
| `k` / `↑` | 1行上スクロール |
| `d` | 半ページ下 |
| `u` | 半ページ上 |
| `h` / `←` / `PageUp` / `Ctrl b` | ページ上 |
| `l` / `→` / `PageDown` / `Ctrl f` | ページ下 |
| `s` | 検索モードに入る |
| `e` | エディタでスクロールバック編集 |
| `Ctrl c` | 最下部に戻り Normal へ |
| `Ctrl s` | Normal モードに戻る |

## Search (Scroll モード内で `s` → 検索語入力 → `Enter`)

| Key | Action |
|-----|--------|
| `n` | 次の検索結果 |
| `p` | 前の検索結果 |
| `c` | 大文字小文字区別の切替 |
| `w` | 折り返し検索の切替 |
| `o` | 単語一致の切替 |

## Session (`Ctrl o`)

| Key | Action |
|-----|--------|
| `w` | セッションマネージャ |
| `c` | 設定画面 |
| `p` | プラグインマネージャ |
| `s` | セッション共有 |
| `a` | About |
| `d` | デタッチ |
| `Ctrl o` | Normal モードに戻る |
